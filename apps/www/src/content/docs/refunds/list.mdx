---

В этом разделе мы рассмотрим, как получить список всех возвратов. Этот функционал полезен для отображения истории возвратов.

<Steps>

### Вызов метода для получения списка возвратов

Метод getRefunds позволяет получать список всех возвратов. Вы можете использовать его для фильтрации возвратов по датам, а также задавать ограничение на количество отображаемых возвратов на странице (пагинация). Этот метод возвращает массив объектов с деталями возвратов.

</Steps>
