title: Управление программами и проектами
description: Руководство по созданию и редактированию программ и проектов
В этом разделе описаны шаги по созданию и редактированию программ, которые состоят из проектов. Программа — это объект управления, объединяющий несколько проектов. Например, программа «ЖЦ Битцевские холмы» может включать проекты «Корпус 1», «Корпус 2» и «Котельная».

<Steps>
Создание программы
На главном экране нажмите кнопку «+ Добавить программу».

В открывшейся форме заполните следующие поля:

Название программы (обязательное поле) — укажите название программы, например, «ЖЦ Битцевские холмы».

Описание (не обязательное поле) — добавьте детальную информацию о программе, например, «Комплекс жилых зданий с инфраструктурой».

Перейдите к следующему шагу — загрузке фотографии объекта.

Загрузка фотографии
Фотографию объекта можно добавить двумя способами:

Кнопка «Камера» — сделайте фотографию объекта непосредственно через приложение.

Кнопка «Фотографии» — выберите изображение из вашей галереи.

После заполнения всех полей и загрузки фотографии кнопка «Создать» станет активной. Нажмите её, чтобы завершить создание программы.

Редактирование программы
Для перехода в режим редактирования программы найдите нужную программу на главном экране.

Нажмите кнопку «…», расположенную на изображении программы.

Откроется форма редактирования, которая идентична форме создания программы.

Примечание:
Форма редактирования программы полностью повторяет форму создания. Вы можете изменить название, описание или фотографию программы.

После внесения изменений нажмите кнопку «Сохранить», чтобы обновить данные.

</Steps>
