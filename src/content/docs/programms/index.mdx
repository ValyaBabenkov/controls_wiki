В этом разделе описаны шаги по созданию и редактированию программ, которые состоят из проектов.

Программа — это объект управления, объединяющий несколько проектов.
Пример структуры программы

```typescript
Программа: ЖЦ Битцевские холмы
  ├── Проект: Корпус 1
  ├── Проект: Корпус 2
  └── Проект: Котельная
```

## Создание программы

<Steps>

### Атрибуты
1. На главном экране нажмите кнопку «+ Добавить программу».
2. В открывшейся форме заполните следующие поля:
- Название программы (обязательное поле) — укажите название программы, например, «ЖЦ Битцевские холмы».
- Описание (не обязательное поле) — добавьте детальную информацию о программе, например, «Комплекс жилых зданий с инфраструктурой».
- Перейдите к следующему шагу — загрузке фотографии объекта.

### Загрузка фотографии
Фотографию объекта можно добавить двумя способами:
1. Кнопка «Камера» — сделайте фотографию объекта непосредственно через приложение.
2. Кнопка «Фотографии» — выберите изображение из вашей галереи.
</Steps>

После заполнения всех полей и загрузки фотографии кнопка «Создать» станет активной. Нажмите её, чтобы завершить создание программы.

## Редактирование программы
1. Для перехода в режим редактирования программы найдите нужную программу на главном экране.
2. Нажмите кнопку «…», расположенную на изображении программы.
3. Откроется форма редактирования, которая идентична форме создания программы.
После внесения изменений нажмите кнопку «Изменить», чтобы обновить данные.


## Удаление программы
1. Для удаления программы найдите нужную программу на главном экране.
2. Нажмите кнопку «…», расположенную на изображении программы.
3. В меню выберите кнопку «Удалить».
4. Подтвердите удаление программы.

<span className={'text-error'}>`Важно: Обратите внимание, что при удалении программы все данные, связанные с ним, будут безвозвратно удалены.
	Восстановление данных будет невозможно.
	Убедитесь, что вы действительно хотите удалить программу перед подтверждением действия.`</span>
