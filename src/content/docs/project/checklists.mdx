Система предоставляет возможность загружать собственные чек-листы или выбирать готовые шаблоны из встроенной библиотеки.

## Добавление чек-листа

<Steps>

### Атрибуты
1. Перейдите в раздел «Шаблоны чек-листов» и нажмите кнопку «Добавить чек-лист».
2. В открывшейся форме заполните следующие поля:
- Название чек-листа (обязательно к заполнению)

### Выбор чек-листа
Выберите один из трех вариантов:
- <b>Выбрать из библиотеки</b> – используйте готовый шаблон из встроенной библиотеки.
- <b>Загрузить свой</b> – загрузите собственный файл чек-листа.
- <b>Скачать шаблон</b> – скачайте шаблон, заполните его и загрузите обратно.

### Шапка документа
Настройте верхнюю часть документа для выгрузки в формате PDF

> Примечание: поле можно оставить пустым

### Список согласующих
Укажите список лиц, который будет отображаться в конце документа при выгрузке в формате PDF.

</Steps>

После заполнения всех полей кнопка «Создать» станет активной. Нажмите её, чтобы завершить добавления.

## Редактирование содержания чек-листа

Чтобы отредактировать содержимое чек-листа:
1. Нажмите на выбранный чек-лист в списке.
2. В открывшемся окне вы можете:
- Скорректировать параметры чек-листа.
- Удалить ненужные разделы или пункты.
4. Сохраните изменения.

> Примечание: форма редактирования идентична форме создания помещения.

## Удаление чек-листа

Чтобы удалить помещение из проекта:
1. Перейдите в раздел «Справочник помещений».
2. Нажмите на элемент, который хотите удалить.
3. В открывшемся меню выберите «Удалить».
4. Подтвердите операцию.
