Раздел «Команда» позволяет взаимодействовать с участниками проекта: исполнителями, заказчиками и партнерами в едином информационном пространстве.

Для просмотра списка участников команды перейдите в проект и откройте вкладку «Команда».

## Добавление пользователя в проект

> Важно: Пользователь должен быть зарегистрирован в системе и иметь активную учетную запись.

<Steps>

### Атрибуты
1. Нажмите кнопку «Добавить участника» в разделе «Команда».
2. В открывшейся форме заполните следующие поля:
- Email - укажите адрес электронной почты пользователя, которого хотите добавить.
- Нажмите кнопку «Далее»

### Роль в проекте
Выберите роль пользователя в проекте из списка.

Если нужной роли нет, обратитесь в адрес <a className={'text-link'} href={"mailto:valya_babenkov@mail.ru?subject=Техническая поддержка"}>технической поддержки</a> с предложением добавить новую роль.

### Функциональные возможности
Назначьте уровни доступа пользователю
- Просмотр информации проекта – доступ к просмотру данных проекта.
- Редактирование информации в проекте – возможность управления проектом (создание, изменение и удаление данных). Нажмите кнопку «Создать».
Нажмите кнопку «Создать».

</Steps>

На указанный email будет отправлено приглашение.

## Активация приглашения
Пользователь, получивший приглашение, должен:

1. Открыть приложение.
2. На главном экране нажать кнопку «Присоединиться к проекту».
3. В открывшемся окне ввести код подтверждения из письма.

После успешной активации проект появится в списке доступных у пользователя.

> Важно: Код приглашения действителен в течение 24 часов.

## Удаление пользователя из команды
Чтобы удалить пользователя из команды проекта:

1. Перейдите в раздел «Команда».
2. Нажмите на пользователя, которого хотите удалить.
3. В открывшемся меню выберите «Удалить».
4. Подтвердите операцию.

После успешного удаления пользователь будет исключен из проекта.
