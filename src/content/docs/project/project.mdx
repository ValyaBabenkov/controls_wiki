В этом разделе описаны шаги по созданию, редактированию и удалению проектов, которые содержатся в программе.

Проект — это объект управления, который включает функциональные модули (команды, осмотры, чек-листы, справочник помещений).

Примеры проектов:
```typescript
Проект: Корпус 1
Проект: Реконструкция котельной
Проект: Ремонт квартиры 259
```

## Создание проекта

<Steps>

### Атрибуты

1. Для создания проекта необходимо перейти в программу и нажать кнопку «+ Добавить проект».
2. В открывшейся форме заполните следующие поля:
- Название проекта (обязательное поле) — укажите название проекта, например, «Ремонт квартиры 259».
- Адрес (необязательное поле) — добавьте адрес проекта, например, «г. Москва, ул. Перово».
- Описание (необязательное поле) — добавьте детальную информацию о проекте, например, описание целей или задач.
- Перейдите к следующему шагу — загрузке фотографии объекта.

### Загрузка фотографии
Фотографию объекта можно добавить двумя способами:
1. Кнопка «Камера» — сделайте фотографию объекта непосредственно через приложение.
2. Кнопка «Фотографии» — выберите изображение из вашей галереи.

### Локация
Можно указать привязку проекта к карте.

</Steps>

После заполнения всех полей кнопка «Создать» станет активной. Нажмите её, чтобы завершить создание проекта.

## Редактирование проекта
1. Для перехода в режим редактирования проекта перейдите в карточку проекта.
2. Нажмите кнопку «…», расположенную в верхнем правом углу.
3. В меню выберите кнопку «Изменить».
4. Откроется форма редактирования, которая идентична форме создания проекта.
5. После внесения изменений нажмите кнопку «Изменить», чтобы обновить данные.


## Удаление проекта
1. Для удаления проекта перейдите в карточку проекта.
2. Нажмите кнопку «…», расположенную в верхнем правом углу.
3. В меню выберите кнопку «Удалить».
4. Подтвердите удаление проекта.

<span className={'text-error'}>`Важно: Обратите внимание, что при удалении проекта все данные, связанные с ним, будут безвозвратно удалены.
	Восстановление данных будет невозможно.
	Убедитесь, что вы действительно хотите удалить проект перед подтверждением действия.`</span>
